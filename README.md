# ConwaysTM
Conway's game of life in a turing machine using the [STEM](https://github.com/andreicozma1/STEM) turing machine simulator

# Usage
1. Run the `STEM.jar`
2. Click `Load Machine` and select `TM.txt`
3. Place a valid board (described below) into the tape
4. Run the machine

# Helper Scripts
`helper_scripts` contains useful scripts used in the creation and usage of this project
- `merger.py` - takes two saved Turing Machine files generated by STEM and combines them into one
  - Usage: `./merger.py [file1] [file2]`
- `rendering.py` - takes a save generated by this Turing Machine and creates a render of the board state in command line
  - Usage: `./rendering.py [file1]`

# Input Format
- Board must be rectangular
- Input 2d board in a flatted 1d structure, row by row
- Starts with a blank, followed by `S`
- Ends with an `X`
- Between each row put a `#`
- Dead cells are `-`
- Live cells are `+`

Example board: 6x6
```
S
--++--#
++--++#
--++--#
++--++#
--++--#
++--++X
```
